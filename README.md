# Portfolio Website

Bu loyiha — shaxsiy **portfolio sayti** bo‘lib, zamonaviy va moslashuvchan dizayn asosida yaratilgan.  
Unda quyidagi bo‘limlar mavjud: 

- **Sidebar (chap panel):** Profil rasmi, ism va qisqa ma’lumot.  
- **About Section:** O‘zingiz haqingizda qisqacha tavsif.  
- **Skills Section:** Texnologiyalar va ko‘nikmalar ro‘yxati.  
- **Projects Section:** Loyihalar taqdimoti va havolalar.  
- **Contact Section:** GitHub, Telegram va boshqa aloqalar tugmalari.  

## Texnologiyalar
- **HTML5**  
- **CSS3 (responsive design)**  
- **Flexbox & Grid**  

## Xususiyatlar
- To‘liq **responsive** (mobil qurilmalar uchun moslashuvchan).  
- Gradient fon, soyalar va hover animatsiyalari.  
- Minimalistik, toza va chiroyli UI.
